COLORS

        
        색상명: blue;
        hex(헥사코드) : #0000ff; = #00f; 로 작성해도 같은 결과
        rgb : rgb(0,0,255);
        rgb+투명도 : rgba(0,0,255,0.5);
        hsl(색조 채도 명도) : hsl(240, 100%, 100%); -> hue, saturation, lightness
        hsl+투명도 : hsla(240, 100%, 100%, 0.5);
